# Vim

1、编辑多文件

```bash
:n #编辑下一个
:N #编辑上一个
```

2、多窗口编辑文件

```bash
:sp file #打开file，上下显示
vim -ON file1, file2 #左右显示
vim -oN file1, file2 #左右显示
```

3、在vim中执行shell命令

```bash
:!command
```

4、滚动屏幕

```bash
ctrl + e #向下滚动一行
ctrl + y #向上滚动一行
ctrl + d #向下滚动半页
ctrl + u #向上滚动半页
ctrl + f #向上滚动一页
ctrl + b #向上滚动一页
```
