# 汇编指令说明

1、leave

```assembly
; 含义：回到caller函数的栈帧
mov esp, ebp
pop ebp
```

2、ret

```assembly
; 含义：弹出返回地址给eip，回到caller函数继续执行
; 机器码：0xC3
pop eip
```

3、call

```assembly
; 含义：保存返回地址，跳转到callee函数
push eip
jmp ******<callee>
```

